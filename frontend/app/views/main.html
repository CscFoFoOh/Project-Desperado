Welcome to Project Desperado!

Project Desperado is a collaboration tool for developers of all skill levels.

Here's how it works:

Users can post a project that they are looking for help on.

Other users can search for projects in their area of interest and apply to be a part of them.

Anyone can see project and user info in order to view a project they are interested in, or a developer's skillset.
